# Chien 資料夾雲端備份流程

```bash
# Step 1 — 在 feature/Chien/graph-queries 先備份
git add Chien/
git commit -m "docs: update planning notes"
git push

# Step 2 — 同步到 Chien branch
git checkout Chien
git checkout feature/Chien/graph-queries -- Chien/
git add Chien/
git commit -m "docs: sync Chien folder"
git push

# Step 3 — 切回工作 branch
git checkout feature/Chien/graph-queries
```
